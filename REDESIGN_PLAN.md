## 四、首页布局设计（自上而下）

### 4.1 导航栏 (Fixed Top)
```
┌─────────────────────────────────────────────────────┐
│  悬象                    却顾所来径...    🌓  中/EN  │
└─────────────────────────────────────────────────────┘
```
- 左侧：品牌名「悬象」（书法风格字体 or 特殊 CSS 处理）
- 中间：签名诗句（小字淡灰，desktop 显示，mobile 隐藏）
- 右侧：主题切换图标（☀️🌤🌙 三档循环）+ 中/EN 语言切换

### 4.2 活动横幅 (Compact Banner)
```
┌─────────────────────────────────────────────────────┐
│  ✨ InfiniteSkill v2.0 已上线 — 立即体验 →          │
└─────────────────────────────────────────────────────┘
```
- **紧凑单行设计**，高度约 44px
- 渐变底色 + 一句话 + CTA 链接
- 可关闭（点 × 隐藏），不再占用大面积空间

### 4.3 栏目切换器 (Tab Navigator)
```
┌──────────────────────────────────────────┐
│   🏺 器 · 工具  │  📜 卷 · 书籍  │  🎯 弈 · 游戏  │
└──────────────────────────────────────────┘
```
- 水平排列的 Tab 选项卡
- 点击切换下方内容面板，**不做页面跳转**
- 当前选中 Tab 底部有水墨晕染效果的下划线

### 4.4 内容面板 (Content Panel)
每个 Tab 对应一个面板，面板中是该栏目下的**项目封面卡片**：

```
┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
│  [封面图/图标]   │  │  [封面图/图标]   │  │  [封面图/图标]   │
│                 │  │                 │  │                 │
│  InfiniteSkill  │  │  飞剑弹珠       │  │  (更多项目...)   │
│  智能文档编译器  │  │  Canvas 弹珠游戏 │  │                 │
│                 │  │                 │  │                 │
│  [简介文字...]   │  │  [简介文字...]   │  │  [简介文字...]   │
│  → 进入 ←       │  │  → 进入 ←       │  │  → 进入 ←       │
└─────────────────┘  └─────────────────┘  └─────────────────┘
```
- 卡片展示：封面图标/缩略图 + 项目名 + 一句话介绍 + 进入按钮
- 点击整张卡片或"进入"按钮 → 跳转到对应子页面
- 空栏目显示优雅的「敬请期待」占位

## 五、中英文双语方案

- 使用 JS 对象存储所有文案的中英文版本
- `<html lang="zh-CN">` / `<html lang="en">` 切换
- 点击导航栏的「中/EN」按钮即时切换全站文字
- 使用 `localStorage` 记住用户语言偏好
- 签名英文版：*"Looking back upon the path I came, the blue-green haze stretches far."*

```js
const i18n = {
  zh: {
    brand: '悬象',
    motto: '却顾所来径，苍苍横翠微',
    tabs: { tools: '器 · 工具', books: '卷 · 书籍', games: '弈 · 游戏' },
    // ...
  },
  en: {
    brand: 'Xuán Xiàng',
    motto: 'Looking back upon the path I came, the blue-green haze stretches far.',
    tabs: { tools: 'Tools', books: 'Books', games: 'Games' },
    // ...
  }
};
```

## 六、子页面风格适配

所有子页面（游戏、工具等）需要：

1. **统一顶部导航条**：与主站一致的「悬象」导航 + 返回主页按钮
2. **色彩继承**：读取用户选择的主题（浅/中/深），应用相同的 CSS 变量
3. **共享底纹**：同样的山水 SVG 底纹

### 具体改造项：
- `game.html`：添加统一导航条 + 主题色适配 + 返回按钮
- `infiniteskill/`：在 React 应用中引入主站主题 CSS 变量，适配色彩体系

## 七、技术实现要点

| 要点 | 方案 |
|------|------|
| 主题切换 | CSS 自定义属性 `[data-theme="light/medium/dark"]`，JS 切换 `data-theme` 属性 |
| 语言切换 | JS i18n 对象 + `data-lang` 属性 + `localStorage` 持久化 |
| Tab 切换 | 纯原生 JS，显示/隐藏对应面板，带淡入动画 |
| 底纹纹理 | 内联 SVG 或 CSS `background-image` 绘制的远山剪影 |
| 字体方案 | 中文：思源宋体/Noto Serif SC；英文：Inter；等宽：JetBrains Mono |
| 毛玻璃效果 | `backdrop-filter: blur(12px)` + 半透明背景 |
| 响应式 | Mobile First，Tab 在小屏幕变为可横滑的选项 |
| 构建兼容 | 保持原有 `build.mjs` Monobuild 体系不变 |
