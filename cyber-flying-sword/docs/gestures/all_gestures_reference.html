🌊 挥鞭操控

移动 + 攻击（默认状态）
👍 拇指 任意 ☝ 食指 伸展 🖕 中指 任意 💍 无名指 任意 🤙 小指 任意
检测逻辑：
非握拳 且 非一指定江山 → 默认为挥鞭状态
extendedCount >= 1 && !isFist && !isUltimate

操控映射：
· 食指尖(lm[8])位置 → 鞭子握柄端
· 手指运动速度 → 鞭尾速度 → 伤害
· fingerHistory[N] lerp链 → 鞭体物理

🛡️ 握拳防御

护盾 / 精防
👍 拇指 弯曲 ☝ 食指 弯曲 🖕 中指 弯曲 💍 无名指 弯曲 🤙 小指 弯曲
检测逻辑：
extendedCount <= 1 && !isIndexUp && !isMiddleUp

普通格挡：持续握拳 → 消耗护盾值抵挡伤害
精防(Perfect Block)：
在敌人攻击命中前 ~100ms 内握拳
→ 不消耗护盾 + 大量怒气

代价：握拳期间无法挥鞭（停止移动和攻击）

⚡ 一指定江山

怒气大招 · 全屏飞剑齐射
👍 拇指 伸展 ☝ 食指 伸展 🖕 中指 弯曲 💍 无名指 弯曲 🤙 小指 弯曲
检测逻辑：
isThumbUp && isIndexUp && !isMiddleUp && !isRingUp && !isPinkyUp
angle(thumb_dir, index_dir) ∈ [80°, 100°]

触发条件：
· 手势保持 >300ms（防误触）
· 怒气值已满

效果：全屏hit-stop → 满屏飞剑射向所有敌人
手势 游戏动作 核心检测 备注
🌊 自然手型 挥鞭（移动+攻击） 非握拳 且 非大招 默认状态，手指位置=鞭子根部
✊ 握拳 护盾/格挡 所有手指弯曲 精防窗口≈100ms，放弃移动和攻击
👆 一指定江山 怒气大招 拇指+食指90°，其余弯曲 保持300ms触发，需怒气满
