# 🔥 StoryForge · 故事熔炉

> **AI 辅助小说创作工作台**，男频网文长篇/短篇优先，正在拓展女频与多题材。
> 纯前端 React + IndexedDB，所有数据本地存储，**支持多 AI 厂商**（OpenAI/Claude/Gemini/DeepSeek/通义/豆包/文心/Kimi/智谱/MiniMax/Ollama 等）。

---

## ✨ 一句话价值主张

> **「让 AI 当你的创作助理，但你说了算。提示词全可见、可调、可保存。」**

市面上多数 AI 写作工具把提示词锁死在后端，用户只能用预设套路。StoryForge 把整个提示词系统开放给作者：

- **看得见**：每个 AI 按钮背后的 prompt 一目了然
- **改得动**：参数滑块 / 文字改写 / 临时覆盖，三种调整粒度
- **存得住**：满意了一键「另存为我的版本」，下次直接用
- **跑得连**：链式工作流一键跑完"故事核心 → 世界观 → 角色 → 大纲 → 第一章"

## 🧩 核心模块详解

### 📚 著作信息

#### 项目概况
- 著作名称、流派标签（**77 条标签**，按起点 / 纵横 / 晋江三大平台分类）
- 著作简介、目标字数（最高 500 万字）
- 著作状态（构思中 / 连载中 / 暂停 / 已完结）
- 状态栏：自动统计创建日期、连续创作天数、最近更新

#### 参考书目
- 「故事参考」「风格参考」分类
- 用于 AI 生成时作为隐式风格指引

---

### 🌍 设定库

#### 世界观

把传统"一大段世界观"拆成 **3 个独立子模块**，每模块多个字段，**每个字段都有独立 AI 一键生成按钮 + 调参浮窗**：

**世界起源（3 字段）**
- 🌌 世界来源：创世神话 / 科技起源 / 文明诞生
- ⚡ 力量层次：修真等级 / 魔法体系 / 科技层级
- 🌟 神明设定（复合）：是否存在神明 + 层级 + 名号 + 规则

**自然环境（11 字段）**
- 🌐 世界结构 / 📐 世界尺寸 / 🗺 大陆分布 / 📏 区域面积
- ⛰ 山川河流 / 🌦 分区域气候
- 🌿 自然资源（4 子字段）：珍禽异兽 / 灵药 / 矿石 / 其他特产

**人文环境（7 字段）**
- 📜 世界历史线 / 📅 大事记 / 🧬 种族
- ⚔ 势力分布 / 🏛 政治经济文化 / 🔥 矛盾冲突 / 🗡 道具设计

**AI 调用智能 context**：填写其他字段后，点 AI 生成时会自动把已填字段作为上下文传给 AI，保证内部逻辑自洽。

#### 故事设计

7 字段全部 AI 生成 + 调参：
- 📜 一句话故事（logline）
- 💡 故事概念（high concept）
- 🎯 故事主题
- ⚔️ 核心冲突
- 📊 故事模式（线性 / 莲花地图 / 多线并行 / 蒙太奇）
- 🛤 故事主线 / 🎼 故事复线

#### 角色设计

**6 种 role 分级展示**：

| Role | 视图 | 适用场景 |
|---|---|---|
| 主角 / 重要配角 | 大卡片表（全字段） | 弧光完整、心理戏多的核心人物 |
| 次要角色 | 小卡片 3 列网格 | 戏份不少但不主导的配角 |
| NPC | 紧凑列表（姓名+地点+一句话） | 常驻但不出场的角色 |
| 路人 | 5 列表格（姓名/出场/章节/作用/结局） | 一笔带过的群像 |

**关系网**：力导图可视化角色间关系（亲属 / 师友 / 敌对 / 暧昧 / 合作 / 竞争）

---

### ✏️ 创作区

#### 创作规则
- 写作风格 / 叙事视角（4 种 POV）/ 基调氛围
- 禁忌列表 / 一致性规则 / 参考作品
- **3 字段「AI 建议」按钮**：基于项目类型 + 世界观自动建议规则

#### 大纲
- 卷 / 章节树状结构
- **AI 卷级生成**：基于世界观 + 故事核心，自动生成 3-5 卷概要
- **AI 章节大纲**：把单卷展开为 15-25 章
- 接 PromptRunPanel：滑块调节"节奏 / 卷数 / 每卷章节数"

#### 细纲（独立模块）
- 把单章大纲拆成 **3-6 个场景（scenes）**
- 每个场景：标题 / 概要 / 主要人物 / 地点 / 核心冲突 / 节奏（慢/中/快/高潮）/ 估算字数
- **AI 一键拆场景**：调用 `detail.scene` 提示词

#### 章节列表
- 跨卷的全部章节扁平视图
- 状态徽章（仅大纲 / 初稿 / 已修改 / 已润色 / 定稿）
- 字数统计 / 跳转编辑

#### 正文（富文本编辑器）
- TipTap-based，支持加粗 / 斜体 / 标题 / 列表 / 引用 / 分隔线
- 自动保存（onBlur）
- **5 个 AI 操作**：写正文 / 续写 / 润色 / 扩写 / 去 AI 味
- 字数实时统计

#### 伏笔
- 10 类型（契诃夫之枪 / 预言暗示 / 象征 / 角色 / 对话 / 环境 / 时间线 / 红鲱鱼 / 平行 / 回调）
- 状态流（计划中 → 已埋设 → 已呼应 → 已回收）
- **AI 建议**：基于世界观+角色批量建议 3-12 个伏笔
- 双视图（列表 / 看板）

### 🎯 提示词库 ⭐ 核心特色

这是 StoryForge 与其他 AI 写作工具的**最大差异化**。

#### 模板 Tab

**43 条 system 内置模板**，覆盖 7 大模块、19 种功能：

- **通用包**（19 条）：世界观/角色/大纲/章节/伏笔/故事/规则/细纲/地理/导入
- **仙侠包**（5+1 条）：飞升体系、典雅文笔、心境优先
- **言情包**（5+1 条）：心理戏、CP 张力、双视角
- **现实主义包**（5+1 条）：日常感、克制、不回避琐碎
- **悬疑推理包**（5+1 条）：信息控制、不可靠叙事、章末反转

**每条模板可见**：
- System Prompt（完整可见，不再黑箱）
- User Prompt 模板（含 `{{var}}` `{{#if var}}` 占位符）
- 变量列表
- **可调参数**（select / slider / number / text / boolean，每个可关闭）
- **示例 / 反例**（few-shot，AI 自动生成 + 用户使用中标记）
- 实时预览（用样例变量字典渲染最终 prompt）

**用户操作**：
- 系统模板锁定，但可一键「克隆为我的」编辑
- 我的模板可改 system / user 模板 / 变量 / 参数 / moduleKey
- 「设为激活」让某模板成为对应模块的当前模板
- 「导出」生成 JSON / 「导入」批量导入他人分享的模板

#### 工作流 Tab

**链式编排**，借鉴蛙蛙写作的差异化卖点：

**3 条内置工作流**：

| 工作流 | 步骤 | 用途 |
|---|---|---|
| 极速起书 · 通用（默认）| 5 步：故事核心 → 世界起源 → 主要角色 → 卷大纲 → 第一章正文 | 从零起书 |
| 单章深度生成 | 4 步：拆场景 → 写正文 → 润色 → 去 AI 味 | 精雕细琢一章 |
| 伏笔体系搭建 | 2 步：世界观摘要 → 伏笔建议 | 中期补伏笔 |

**Runner 体验**：
1. 点「运行」→ 顶部「开始 / 中止 / 继续 / 返回」
2. 每步独立卡片（5 种状态色：灰/蓝/绿/红/灰）
3. 流式输出实时渲染
4. `userConfirmRequired` 步骤完成后自动暂停等审核
5. 可「重新生成」「跳过」「📋 复制」「💾 保存到 [字段]」

**自动写回**（saveTarget）：
- 单字段：worldview / storyCore / creativeRules 任意字段
- 批量：⚡ 角色库（AI 输出 JSON 数组）/ 大纲节点 / 伏笔库

**用户自建工作流**：
- 「+ 新建」直接进编辑器
- 可改名 / 描述 / 步骤增删上下移 / 选 moduleKey / 选 saveTarget / 编辑 userHint
- 导出 JSON 分享 / 导入他人工作流

### ⚙️ 设置区

#### 版本历史

- 时间线视图，每条显示标签 / 时间 / 大小

#### 导出
- 项目全数据导出 JSON（可在另一台设备导入恢复）

#### 设置
- **AI 配置**：11 个厂商
  - 国际：OpenAI (GPT) / Anthropic Claude / Google Gemini / DeepSeek

- 测试连接按钮
- **主题切换**：5 个主题（midnight / ocean / graphite / mist / parchment）

### 关键设计：提示词渲染引擎

```ts
renderPrompt(template, ctx, {
  parameterValues?: Record<string, unknown>,  // 运行时参数
  overrides?: { systemPrompt?, userPromptTemplate? }, // 文本覆盖
})
```

支持的语法：
- `{{var}}` — 字符串替换
- `{{#if var}}...{{/if}}` — 条件块（var 真值非空时保留）
- `{{#if usesXxx}}...{{/if}}` — 参数启用条件（用户关闭某参数时块隐藏）
- `{{#if notUsesXxx}}...{{/if}}` — 参数禁用条件（提供默认 fallback）

few-shot 自动拼接：
```
[user prompt]
... ↑ 用户编辑的模板内容

---
【参考示例】
[好示例 1] / [好示例 2] / [好示例 3]
[反例 1] / [反例 2]
```

## 🎓 这个工具适合谁

✅ **网文作者** — 提示词全透明，可调风格，可保存自己的"个人方言"
✅ **新手** — 用工作流"一键起书"，跟着 AI 学结构
✅ **老手** — 把自己已有的 prompt 模板导入，配题材包，效率倍增
✅ **多题材作者** — 切换题材包瞬间换风格，同时维护多部书
✅ **需要 vendor lock-in 自由** — 11 个 AI 厂商任选，本地存储不被绑架

❌ **追求"一键交付完美书"** — 这是工作台不是托管服务，AI 是助理不是替身
