## 1. 游戏概述

### 1.1 高概念 (High Concept)
赛博飞剑是一款**摄像头手势操控**的 2.5D 横版动作游戏。玩家通过挥动右手控制一把由飞剑组成的鞭子，鞭子既是武器也是唯一的移动工具。角色挂载在鞭子上，被鞭子的物理运动带动飞行、战斗。

### 1.2 核心幻想 (Core Fantasy)
**"御剑飞行"的修仙幻想 + 抽陀螺的暴力快感。**
你不是在按键移动一个角色——你是在挥舞一条由百把飞剑组成的光鞭，你的角色只是挂在这条鞭子上的一个存在。你甩鞭子抽打怪物，也甩鞭子把自己抽飞出去。

### 1.3 独特卖点 (USP)
1. **手势即物理**：手的真实运动直接映射为游戏物理力，不是"用手势替代按键"
2. **操控对象不是角色**：你操控的是武器，角色是武器的附属品
3. **单手操控的深度**：只需三个手势（挥鞭/握拳/一指定江山），但组合出极高的策略深度

### 1.4 目标平台
- Web 浏览器（Chrome/Edge/Firefox，需摄像头权限）
- 可选未来：Electron 桌面打包

### 1.5 目标受众
- 喜欢动作游戏、追求操作深度的玩家
- 对新奇交互方式感兴趣的玩家
- 修仙/武侠题材爱好者

## 2. 核心机制

### 2.1 飞剑鞭子（Sword Whip）

#### 物理模型
鞭子是一条由 N 个节点组成的 **lerp 链**（弹簧-阻尼绳体）：

```
手指(食指尖) → [节点0] → [节点1] → [节点2] → ... → [节点N] (鞭尾)
                 ↑高响应      ←—— lerp 系数逐渐降低 ——→      ↑低响应/高延迟
```

- **节点0**（握柄端）：高 lerp（≈0.8），紧跟手指位置
- **节点N**（鞭尾端）：低 lerp（≈0.2-0.35），大幅延迟，产生甩鞭效果
- 每个节点还有轻微的**上漂力**（模拟气场/仙气）

> 这直接复用现有 demo 中的 `fingerHistory[40]` 系统，只是语义从"手指轨迹"变成"鞭子绳体"。

#### 视觉表现
- 120 把飞剑沿鞭子节点排列，形成**剑链鞭**
- 使用 `InstancedMesh`（DynamicDrawUsage）渲染
- 每把剑有发光材质（emissive cyan）+ 尾迹粒子（ShaderMaterial Points）
- 挥鞭速度越快，发光越强，尾迹越长

#### 伤害衰减
鞭子的伤害沿长度方向衰减：

```
伤害(节点i) = D_base × V_segment(i) × (i / N)
```

- `D_base`：基础伤害系数
- `V_segment(i)`：该节点当前帧的速度
- `i / N`：位置权重（尾端=1.0 最高，根部=0 最低）

**鞭尾速度通常是手指速度的 3-5 倍**（真实鞭子物理），所以尾端伤害极高。

### 2.2 角色挂载系统

#### 跟随态 (Attached)
- 角色"挂"在鞭子链的某个中前段节点（如节点 5-10）
- 角色位置 = 该节点位置 + 重力偏移
- 鞭子慢速移动时，角色平稳跟随
- 鞭子快速运动时，角色被甩动

#### 弹射态 (Launched)
- 当甩鞭速度导致角色受到的加速度超过**脱离阈值**时，角色脱离鞭子
- 角色获得脱离瞬间的速度向量，进入**惯性飞行**
- 飞行物理：
  - 惯性运动（保持速度方向）
  - 重力（向下加速度）
  - 空气阻力（速度线性衰减）
  - **画面边缘反弹**（速度保留 70-80%）
- 角色减速到一定阈值后，自动"磁吸"回鞭子（回到跟随态）

#### 气流罩 (Airflow Shield)
当弹射速度超过阈值时：
- 角色周围生成**气流罩**视觉效果（半透明球形 + 风速线）
- **撞到敌人**：对敌人造成伤害 + 自身受伤减少 50%
- **被敌人技能命中**：伤害减少 30%
- 速度降到阈值以下：气流罩消失
- 气流罩是**被动效果**，不消耗任何资源

### 2.3 手势系统

#### 手势 1：挥鞭（默认状态）
- **触发**：手没握拳、不是一指定江山 → 自动进入
- **效果**：手指位置实时驱动鞭子握柄端
- **攻击**：鞭体碰到敌人 hitbox 即造成伤害
- **移动**：鞭子带动角色移动

#### 手势 2：握拳（护盾/格挡）
- **触发**：`extendedCount <= 1 && !isIndexUp && !isMiddleUp`
- **效果**：
  - 鞭子立即收回并围绕角色旋转形成护盾（视觉上飞剑环绕）
  - 吸收伤害，消耗护盾值
  - **角色位置固定**（不再被鞭子带动，悬浮在原地）
  - **无法攻击**
- **精防 (Perfect Block)**：
  - 在敌人攻击命中前 100ms 内从非握拳切换到握拳
  - 不消耗护盾值
  - 获得大量怒气（约为普通被攻击怒气的 5 倍）
  - 视觉：闪光 + hit-stop 50ms + 屏幕震动
  - 音效：金属碰撞 + 回响

#### 手势 3：一指定江山（怒气大招）
- **触发**：
  - 拇指+食指伸展呈 90° 角
  - 中指/无名指/小指弯曲贴掌
  - 保持 300ms 以上
  - 怒气条已满
- **效果**：
  - 全屏 hit-stop（时间冻结 500ms）
  - 鞭子散开为 120 把独立飞剑
  - 每把剑锁定一个敌人（或多把锁定同一个）
  - 齐射！飞剑穿透敌人，造成极高伤害
  - 视觉参考：魔兽争霸 3 的 3c 模式光暗游侠合体大招
  - 怒气清零

### 2.4 资源系统

#### 血量 (HP)
- 最大值：100（可通过升级/装备增加）
- 受到伤害时减少
- 归零 = 死亡 → 从最近存档点重生
- 恢复方式：恢复点（场景中的固定位置）、Boss 掉落、特殊道具

#### 护盾 (Shield)
- 最大值：50
- 握拳格挡时消耗（1点护盾 = 抵挡 1点伤害）
- 精防不消耗
- 恢复方式：自然缓慢回复（3点/秒），击杀回复（待平衡）
- 护盾耗尽后继续格挡 → 伤害直接穿透到血量（减伤50%）

#### 怒气 (Rage)
- 最大值：100
- 获取：
  - 鞭子命中敌人：+2/次
  - 被敌人攻击：+5/次（痛苦转化为力量）
  - 精防：+25/次（核心获取手段）
- 消耗：释放大招（一指定江山）消耗全部怒气
- 不会自然衰减

## 3. 敌人设计

### 3.1 设计原则
- **敌人是动态的**：能移动、能主动攻击、能释放技能
- **敌人不是靶子**：它们会位移到玩家面前劈砍，会释放需要闪避的技能
- **敌人应该配合鞭子机制**：玩家需要利用鞭子的特性（挥击、弹射、气流罩）来应对不同敌人

### 3.2 敌人类型（初版）

| 类型 | 行为 | 玩家应对 |
|------|------|---------|
| **近战杂兵** | 缓慢靠近，近身劈砍 | 鞭击即可击杀，练习鞭击手感 |
| **突进怪** | 快速冲刺到玩家面前 | 需要及时挥鞭击退或弹射闪避 |
| **远程怪** | 固定位置发射弹幕 | 需要利用弹射穿过弹幕接近 |
| **盾怪** | 正面防御，需要从背后攻击 | 利用鞭子的弧线运动绕到背后 |
| **精英怪** | 组合技能，能读取玩家运动趋势 | 需要打精防获取怒气，用大招击破 |
| **Boss** | 复杂攻击模式、多阶段 | 综合运用所有机制 |

### 3.3 敌人攻击类型

| 攻击类型 | 表现 | 应对 |
|---------|------|------|
| **近身斩** | 敌人举刀→劈下（有预警动画） | 挥鞭闪避 或 精防 |
| **突刺** | 敌人冲刺直线攻击 | 垂直方向闪避 |
| **弹幕** | 多个投射物 | 弹射穿越（气流罩减伤） |
| **范围AOE** | 地面标记→爆炸 | 弹射离开区域 |
| **追踪技能** | 缓慢追踪玩家的能量球 | 利用反弹甩掉，或高速撞碎（气流罩） |

## 4. 关卡设计

### 4.1 横版卷轴探索
- 2.5D 横版场景（Z 轴深度用于视差和前后层）
- 平台地形：利用鞭子弹射到达高处/远处平台
- 隐藏区域：需要特定弹射角度才能到达
- 恢复点：场景中的固定位置，触碰即回复 HP

### 4.2 Boss 关卡
- 封闭场地（结界/阵法边界）
- 结界壁 = 弹射反弹墙
- Boss 多阶段设计：
  - 第一阶段：学习 Boss 攻击模式
  - 第二阶段：Boss 加速/新技能
  - 最终阶段：需要精防+大招的组合才能击破

### 4.3 场景主题（初步规划）

| 章节 | 主题 | 视觉风格 |
|------|------|---------|
| 第一章 | 暗黑森林 | 深绿色调，雾气，闪电（现有 demo 风格） |
| 第二章 | 赛博都市 | 霓虹灯，建筑，雨夜 |
| 第三章 | 虚空仙境 | 浮空岛屿，星空，灵气粒子 |
| 最终章 | 天劫雷域 | 雷电主题，混沌空间 |

## 5. 视觉与音频

### 5.1 美术风格
- **赛博修仙**：科技感 + 中式修仙元素
- 主色调：青色 (#00FFFF) + 暗底
- 发光效果大量使用（UnrealBloomPass）
- 飞剑材质：蓝色金属 + 青色发光
- 粒子系统：尾迹、爆炸、气流罩

### 5.2 音效设计
- **程序化合成**（Web Audio API）——不依赖音频文件
- 鞭击音：锯齿波 + 快速频率扫描
- 格挡音：金属碰撞（方波 + 低频衰减）
- 精防音：清脆金属音 + 回响（混响节点）
- 雷电音：白噪声 + 低通滤波 + 指数衰减
- 大招音：多层叠加（风声 + 剑鸣 + 爆炸）
- BGM：待定（可以是程序化环境音或外部音频文件）

### 5.3 特效列表

| 特效 | 触发时机 | 表现 |
|------|---------|------|
| **鞭击拖尾** | 挥鞭时 | 发光粒子点尾迹 |
| **命中火花** | 鞭子碰到敌人 | 粒子爆发 + hit-stop 20ms |
| **气流罩** | 弹射速度超阈值 | 半透明球 + 风速线 |
| **精防闪光** | 完美格挡 | 全屏闪白 50ms + 震动 |
| **大招演出** | 一指定江山触发 | hit-stop 500ms → 飞剑齐射 → 多段爆炸 |
| **闪电** | 随机环境 | 点光源闪烁 + 雷声 |
