## 🏛 核心系统架构图纸
整个项目目前是一个以**根目录门户(Portal)**为主将，各类型**子站点**为分舰队的多重架构系统。

1. **主门户站 `/index.html`**：
   - **技术栈**：原生 HTML5 + 极其极致的 Vanilla CSS3 (抛弃臃肿框架)。
   - **视觉标准**：暗黑/琉璃/羊皮纸切换的混合极客视觉（目前是带有径向渐变背景的沉浸风）。
   - **主打组件**：采用原生鼠标捕捉滚动的横向 Carousel 轮播大屏。

2. **第一子应用：智能技能编译器 `/infiniteskill/`**：
   - **功能**：将纯文本抽离提炼成专业的 LLM 结构化系统 Prompt 技能包。
   - **技术体系**：Vite + React (TypeScript) + TailwindCSS v4。
   - **改动标志**：底色专门使用了最高级的“羊皮卷枯黄” (`#F4EED1`)；去掉了繁杂的模型配置输入。

3. **防盗安全网 (暗黑后台代理)**：
   - 具有一个放置在项目根目录 `/api/v2/telemetry-sync.js` 的 **Vercel Serverless Function**。
   - 只要携带假造的请求头 `X-Core-Version: 8192` 向它发请求，它就能利用站长埋在线上的绝密变量 `SECRET_GEMINI_KEY` 白嫖调用。
