## 🎯 这是什么？

**功夫弹珠**是一款融合中国武侠元素的弹珠消砖（Breakout）游戏。你将操控一柄飞剑，在充满剑气的江湖中击碎砖块、守护红心、挑战八大关卡。

整个游戏只有一个 HTML 文件，无任何外部依赖，打开浏览器即可畅玩。

> **以剑为桨，以珠为刃，在像素江湖中打出你的武林传说。**

## 🎮 操作方式

### 桌面端

| 操作 | 按键 |
|------|------|
| 移动飞剑 | 鼠标左右移动 |

| 蓄力发射 | 按住鼠标再松开（按住越久速度越快） |

## 🛠️ 技术实现

### 技术栈

- **HTML5 Canvas** — 全部渲染
- **原生 JavaScript** — 游戏逻辑，ES6+ 严格模式
- **Web Audio API** — 程序化生成所有音效（无音频文件）
- **零外部依赖** — 不依赖任何库或框架

### 游戏引擎特性

| 特性 | 实现方式 |
|------|---------|
| 物理碰撞 | 矩形 + 圆形碰撞检测，反射角度计算 |
| 粒子系统 | 自定义粒子引擎，支持三档密度 |
| 音频系统 | Web Audio API 振荡器 + 噪声合成 |
| 关卡系统 | 程序化生成，支持 8 种不同阵型 |
| 存档系统 | localStorage 保存进度/设置/排行榜 |
| 自适应布局 | 动态计算砖块尺寸，适配任意屏幕 |

### 砖块类型

| 类型 | 血量 | 描述 |
|------|------|------|
| 普通砖 | 1 | 标准砖块，一击即碎 |
| 硬砖 | 2 | 需要两次命中 |
| 钢砖 | 3 | 最坚固的砖块 |
| ❤️ 红心 | 3-5 | 关卡核心目标 |
